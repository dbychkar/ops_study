Список изученных команд Ubuntu:

    clear -  очистить окно терминала
    reset -  сбросить настройки терминала
    whatis  - однострочная справка по командам
    Ключ --help - краткая справка по командам
    man, info - подробная справка по командам
    apropos - поиск по справке
    history - вывод истории команд

    pwd -  (print working directory) вывести путь к текущей папке
    cd -  (change directory) сменить папку
    ls  - вывести содержимое папки
    ll - псевдоним для ls -la

    mkdir -  (make directory) создать новую папку
    tree -  вывести файловую структуру в виде дерева
    touch -  создать файл
    echo  - вывести текст на экран (или отправить в файл)
    cat- (concatenate) вывод "склеенного" содержимого файлов

    cp -  (copy) копировать файл или папку
    mv -  (move) переместить/переименовать файл или папку
    rm -  (remove) удалить файл или папку

    more -  постраничный просмотр содержимого файла
    less -  постраничный просмотр содержимого файла
    head - просмотр первых строк файла
    tail -  просмотр последних строк файла

    df-  отчет об использовании дисков
    du-  вывод информации об использовании дискового пространства 
    tar- архиватор GNU
    zip- сторонний архиватор

    whoami -  вывести имя пользователя (логин)
    id -  вывести UID, GID, логин, список дополнительных групп пользователя
    groups -  вывести список групп пользователя
    su - (substitute user) - сменить пользователя
    sudo - (super user do) исполнить команду от имени суперпользователя

    useradd - создать пользователя (основная команда)
    adduser - создать пользователя (perl-скрипт !!при создании пользователя позволит сразу задать ему пароль)
    passwd - сменить пароль пользователю

    usermod - изменить пользователя
    usermod -L - заблокировать пользователя
    usermod -U - разблокировать пользователя

    visudo - редактировать файл sudoers (права доступа)
    userdel - удалить пользователя

    groupadd - создать группу
    addgroup - создать группу
    groupdel - удалить группу

    chmod - изменить права доступа к файлам
    chown - изменить владельца/группу владельцев файла
    chgrp - сменить группу владельцев
    umask - сменить или вывести маску для создания файлов

    chmod - изменить права доступа к файлам
    chown - изменить владельца/группу владельцев файла
    chgrp - сменить группу владельцев
    umask - сменить или вывести маску для создания файлов
    
    ln - создать жесткую ссылку на файл
    ln -s - создать мягкую ссылку на файл
    ls -li - вывести список файлов в табличном виде, включая номера файлов (iNodes)

    top –o PID - запустить утилиту top с сортировкой по полю PID
    top –u salesguy - запустить утилиту top с фильтрацией процессов по пользователю salesguy


    htop - более мощная утилита top (юзерфрендли)
    ps - вывести инфо о текущем процессе
    ps 1 - вывести инфо о процессе номер 1
    ps aux - вывести снимок состояния всех процессов в виде таблицы
    ps afx - вывести снимок состояния всех процессов в виде дерева
    pidof - найти номер процесса по его имени
    kill - завершить процесс
    systemctl - утилита для управления службами 
    systemctl status ssh - вывести инфо о состоянии службы SSH
    systemctl stop ssh - остановить службу SSH
    systemctl start ssh - запустить службу SSH
    systemctl is-active ssh - проверить: запущена ли служба SSH


    sudo apt-get remove <пакет> - удалить пакет
    sudo apt-get purge <пакет> -  удалить пакет вместе с настройками

    dpkg -l - вывести список установленных пакетов
    dpkg -l | wc -l - вывести количество установленных пакетов
    dpkg –l tree - вывести инфо о пакете tree
    apt show tree - вывести инфо о пакете tree
    apt download htop - скачать пакет в виде *.deb файла
    sudo dpkg -i htop_3.0-11_amd64.deb - установить пакет утилитой dpkg
    sudo apt install snapd - установить пакетный менеджер snap
    apt search opera - поискать пакет opera в репозиториях apt
    snap find opera - поискать пакет opera в репозиториях snap
    snap list - вывести список пакетов, утановленных менеджером snap
    sudo snap install opera - установить пакет opera
    sudo snap remove opera - удалить пакет opera


    systemctl status cron - проверить инфо о службе cron
    sudo systemctl start cron - запустить службу cron
    crontab -l - просмотр списка запланированных команд текущего пользователя
    crontab -e - редактирование списка задач текущего пользователя
    crontab -r - удаление списка задач текущего пользователя
    crontab -u user_name -e - редактировать задачи другого пользователя




ls /dev - вывод содержимого папки /dev
ls /dev/sd* - вывод отфильтрованного содержимого папки /dev
 

whatis df - вывод однострочной справки по команде df
df -h - вывод информации об использовании дискового пространства файловых систем
df -hT - добавление колонки Type (тип устройства) в предыдущую команду
df -hT -t ext4 - фильтрация по колонке Type в предыдущей команде
df -hT -t ext4 -i - вывод информации об используемом месте в inodes
 

whatis du - вывод однострочной справки по команде du
du ~ -s - подсчет занимаемого места домашним каталогом пользователя (~)
du ~ -sh - подсчет занимаемого места домашним каталогом пользователя (~)
du -hc - подсчет занимаемого места текущей папкой
du *.txt -h - подсчет занимаемого места файлами с расширением 'txt'
 

whatis mount - вывод однострочной справки по команде mount
mount - вывод информации командой mount без параметров
 

whatis fdisk - вывод однострочной справки по команде fdisk
whatis gdisk - вывод однострочной справки по команде gdisk
sudo fdisk -l /dev/sda - вывод информации о диске /dev/sda
 

whatis lsblk - вывод однострочной справки по команде lsblk
lsblk - вывод списка блочных устройств
history - вывод истории использованных команд    

Горячие клавиши:

    Ctrl+L - очистить окно терминала
    Ctrl+U - стереть текст слева от курсора
    Ctrl+K - стереть текст справа от курсора


Ctrl + A Перейти в начало командной строки.
Ctrl + E Перейти в конец командной строки
Ctrl + F Переместиться на один символ вперед
Ctrl + B Переместить один символ назад
Ctrl + XX Переключить положение курсора между началом командной строки и текущей позицией.
Ctrl + ] + x Перемещает курсор вперед к следующему вхождению x.
Alt + F / Esc + F Перемещает курсор на одно слово вперед.
Alt + B / Esc + B Перемещает курсор на одно слово назад.
Alt + Ctrl + ] + x Перемещает курсор к предыдущему вхождению x.

Bash-управление/процесс

Ctrl + L Аналогично команде очистки, очищает экран терминала.
Ctrl + S Останавливает вывод команды на экран.
Ctrl + Z Приостанавливает выполнение текущей команды и перемещает ее на задний план.
Ctrl + Q Возобновляет приостановленную команду
Ctrl + C отправляет сигнал SIGI и уничтожает выполняющуюся в данный момент команду.
Ctrl + D Закрывает текущий терминал.

История Баша

Ctrl + R Инкрементный обратный поиск в истории bash
Alt + P Неинкрементный обратный поиск в истории bash
Ctrl + J Завершает поиск в истории по текущей команде.
Ctrl + _ Отменить предыдущую команду
Ctrl + P / Стрелка вверх Переход к предыдущей команде.
Ctrl + N / Стрелка вниз Переход к следующей команде
Ctrl + S Получает следующую последнюю команду.
Ctrl + O Запускает и повторно вводит команду, найденную с помощью Ctrl + S и Ctrl + R.
Ctrl + G Выход из режима поиска по истории
!! Выполняет последнюю команду
!* Выполняет предыдущую команду, кроме ее первого слова.
!*:p Отображает то, что !* заменяет
!x Запускает последнюю команду в истории bash, которая начинается с x.
!x:p Отображает команду x и добавляет ее как последнюю команду в истории.
!$ То же, что и OPTION+., выводит последний аргумент предыдущей команды.
!^ Заменяет первый аргумент последней команды в текущей команде.
!:pОтображаетслово,котороезаменяет!:pОтображаетслово,котороезаменяет!.
^123^abc Заменяет 123 на abc.
!n:m Повторяет аргумент в пределах диапазона (т. е. m 2–3).
!fi Повторяет последнюю команду в истории, которая начинается с fi.
!n Запустить n-ю команду из истории bash
!n:p печатает команду, которую !n выполняет
!n:Повторитьаргументыпоследнейкоманды(т.е.отаргументаnдоПовторитьаргументыпоследнейкоманды(т.е.отаргументаnдо)
Редактирование Bash
Ctrl+U Удаляет перед курсором до начала команды
Ctrl+K Удаляет после курсора до конца команды
Ctrl + W Удаляет команду/аргумент перед курсором.
Ctrl+D Удаляет символ под курсором.
Ctrl + H Удаляет символ перед курсором.
Alt+D Удаляет из символа до конца слова.
Alt + Backspace Удаляет из символа до начала слова.
Альт + . / Esc+. Использует последний аргумент предыдущей команды
Alt + < Переходит на первую строку истории bash.
Alt + > Переход к последней строке истории bash.
Esc + T Переключение между двумя последними словами перед курсором.
Alt + T Переключает текущее слово с предыдущим.
